Elysium Days is a Vanilla+ modpack with no quest books or guides. It expands the game with QOL and new features that Mojang wouldn't add, but fit in perfectly.

![EDDivider](https://cdn.modrinth.com/data/cached_images/fc96a44e33fe08221da632f4003b825f007705f5.png)

# What does this modpack offer?

<summary>⛔ <font color="#de3735">Incompatible Mods</font></summary>

These are the mods that don't work with Elysium Days either because of incompatibilities or because they do the same thing.

- [Sodium](https://modrinth.com/mod/sodium) → Incompatible with **Embeddium** (Sodium on 1.20.1 has significant issues, therefore Elysium Days uses Embeddium to get fewer issues and better performance)

- [VulkanMod](https://modrinth.com/mod/vulkanmod) → Incompatible with the majority of mods in Elysium Days

- [More Mob Variants](https://modrinth.com/mod/more-mob-variants) → Incompatible with **[Let's Do] Meadow**

- [Better Mount HUD](https://modrinth.com/mod/better-mount-hud) → Incompatible with **Stfu**

- [Model Gap Fix](https://modrinth.com/mod/modelfix) → Incompatible with **Stfu**

- [Yeetus Experimentus](https://modrinth.com/mod/yeetus-experimentus) → Incompatible with **Stfu**

- [NarrusYeetus](https://modrinth.com/mod/narrus-yeetus) → Incompatible with **Stfu**

- [Better Night Vision](https://modrinth.com/mod/betternightvision) → Incompatible with **Stfu**

- [Horse Buff](https://modrinth.com/mod/horsebuff) → Incompatible with **HorseMan**

- [HorseInBoat](https://modrinth.com/mod/horseinboat) → Incompatible with **HorseMan**

- [FancyMenu](https://modrinth.com/mod/fancymenu) → Incompatible with **Elysium Core**

- [Merchant Markers](https://modrinth.com/mod/merchant-markers) → Resource Loading Failed Error

- [Better End](https://modrinth.com/mod/betterend) & [Better Nether](https://modrinth.com/mod/betternether) → Critical rendering issues, crashes, and shader issues with epilepsy risks

- [SleepWarp](https://modrinth.com/mod/sleep-warp) → Causes crashes on servers

- [Sounds](https://modrinth.com/mod/sound) → Causes unplayable FPS drops

- [Kaffee's Dual Ride](https://modrinth.com/mod/kaffees_dual_ride) → Incompatible with **Two Players One Horse** & **Pehkui**

- [Let Sleeping Dogs Lie](https://modrinth.com/mod/let-sleeping-dogs-lie) → Incompatible with **Fresh Animations**

- [cat_jam](https://modrinth.com/mod/cat_jam) → Incompatible with **Fresh Animations**

- [FastAnim](https://modrinth.com/mod/fastanim) → Incompatible with **Fresh Animations** & **Exposure**'s selfie stick
